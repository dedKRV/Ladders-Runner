Обзор проекта
----
Ladders Runner — это платформенный шутер, созданный на базе фреймворка Arcade на языке Python. Игроку предстоит пройти через 5 уникальных уровней, сражаясь с врагами, собирая коллекционные карты и избегая смертельных ловучек.

Проект представляет собой полноценную игру с системой сохранений, меню, настраиваемыми персонажами, различными типами оружия и визуальными эффектами.

Особенности
----
Основной геймплей
5 уникальных уровней с разным дизайном и сложностью

3 типа персонажей: Biker, Punk, Cyborg

3 типа оружия с разными визуальными эффектами

Система врагов: враги отслеживают игрока, стреляют, имеют анимации получения урона и смерти

Коллекционные элементы: карты, деньги, которые нужно собирать для открытия выхода

Ловушки и препятствия: уронные тайлы, транспортеры, прыжковые платформы

Технические особенности
----
Система частиц: кровь, пыль от бега, искры от выстрелов, эффекты попадания

Полноценный интерфейс: главное меню, пауза, game over, экран завершения уровня

Система сохранений: база данных SQLite с сохранением прогресса для каждого уровня

Настраиваемые персонажи и оружие: выбор перед началом игры

Звуковое сопровождение: фоновая музыка и звуковые эффекты

Анимации: плавные анимации персонажей, врагов, интерфейса

Игровые системы
----
Здоровье игрока: система из 10 сердец (каждое по 10 HP)

Система звезд: оценка от 1 до 3 звезд за уровень (карты, враги, время)

HUD: отображение здоровья, убитых врагов, собранных денег и карт

Камера: плавное следование за игроком с ограничениями

Курсор: кастомный курсор для взаимодействия с интерфейсом

Технологический стек
----
Язык программирования: Python 3.13

Игровой движок: Arcade 3.3.3

База данных: SQLite3

Вспомогательные библиотеки: Pygame 2.6.1

Механики игры
----
Управление
Движение: WASD или стрелки

Прыжок: W/↑ (только на земле)

Стрельба: ЛКМ

Вкл/выкл оружия: клавиша 1

Пауза: ESC

Система звезд
----
Уровень оценивается по 3 параметрам:

Собраны все карты (обязательно)

Убиты все враги (опционально)

Время прохождения (опционально)

Оценки:
----

★★★ — все условия выполнены

★★ — карты + одно дополнительное условие

★ — только все карты

Прогресс
----
Сохранение позиции, здоровья, собранных предметов

Отдельные сохранения для каждого уровня

Возможность продолжить с любого уровня
